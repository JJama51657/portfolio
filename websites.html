Blog Website Platform

This is a custom-built Flask blog platform featuring:

  • User registration and login with secure password hashing
  • Rich text editor for creating and editing posts
  • Commenting system tied to authenticated users
  • Admin-only post management including creation, editing, and deletion
  • Email contact form integrated with SMTP for notifications
  • Database-driven with SQLAlchemy and Flask-Login for session handling

This project demonstrates full-stack web development skills with Python, Flask, Bootstrap, and database management.
