Bank Data Analysis

This project explores transaction data from a financial institution, analyzing customer behavior, transaction types, and channel usage using Python, Excel, SQL, and Power BI.

Python + Pandas Code

The Python script serves as the core engine for preprocessing and analyzing raw bank transaction data. It begins by reading a CSV file containing transaction records, parsing key date fields for accurate time-based operations. The script performs multi-level aggregation to summarize key metrics at the customer, transaction type, and channel levels. It calculates total and average transaction amounts, identifies the most common occupations per customer, and detects patterns in transaction types (e.g., credit, debit) and usage channels (e.g., ATM, online, in-person). The cleaned and aggregated results are then exported into separate CSV files, making them ready for deeper analysis and visualization in Excel and Power BI. This preprocessing pipeline enables faster insight generation by reducing noise and structuring the data efficiently.

import pandas as pd df = pd.read_csv(r"Bankinfo\\bank_transactions_data_2.csv", parse_dates=["TransactionDate", "PreviousTransactionDate"]) df['TransactionDate'] = pd.to_datetime(df['TransactionDate']) # Customer-Level Summary customer_summary = df.groupby("AccountID").agg({ "TransactionID": "count", "TransactionAmount": ["sum", "mean"], "CustomerAge": "first", "CustomerOccupation": lambda x: x.mode().iloc[0] if not x.mode().empty else None }).reset_index() # Transaction Pattern Table Transaction_chart = df.groupby("TransactionID").first().reset_index() # Transaction Type Summary transaction_type_summary = df.groupby("TransactionType").agg({ "TransactionID": "count", "TransactionAmount": "mean" }).reset_index() transaction_type_summary.columns.values[1] = "Total_Count" total_txn = df.shape[0] transaction_type_summary["Percentage"] = (transaction_type_summary["Total_Count"] / total_txn * 100).round(2) # Channel Summary channel_summary = df.groupby("Channel").agg({ "TransactionID": "count", "TransactionAmount": "mean" }).reset_index() channel_summary.columns = ["Channel", "Count_of_transactions", "Mean_sum_of_transactions"] # Save summaries customer_summary.to_csv("customer_summary.csv", index=False) Transaction_chart.to_csv("Info_Chart.csv", index=False) transaction_type_summary.to_csv("transaction_type_summary.csv", index=False) channel_summary.to_csv("channel_summary.csv", index=False)

Excel Sheet

The cleaned and exported data was opened in Excel, where SQL queries were used to sort and filter key variables. I used Excel tables to rank transaction amounts by customers, regions, and occupations. This helped identify the most valuable customer segments.

Power BI Dashboard

The data was visualized using Power BI dashboards which include:

  • Heatmaps of states with the highest concentration of transactions
  • Occupation-based spending breakdown
  • Comparative graphs of transaction types (e.g. credit vs debit)
  • Mediums used (ATM, online, in-person)

These insights helped identify which demographics and transaction methods contribute most to the business.
