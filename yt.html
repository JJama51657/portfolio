YouTube Data Analysis Project

Analyzing audience engagement and trends across top YouTubers using Python and data visualization.

Project Overview

This project processes YouTube channel data to uncover key insights about audience engagement and popular content categories. It calculates engagement metrics by analyzing average views, likes, comments, and subscriber counts across various YouTubers, and visualizes the results through bar charts, pie charts, and scatter plots.

Main Features & Insights

  • Data cleansing and normalization of view, like, comment, and subscriber counts from CSV files.
  • Calculation of Engagement Activity (%) to highlight YouTubers with highest audience interaction.
  • Aggregation of engagement by content categories to identify top-performing niches.
  • Analysis of engagement rates across countries, showing geographic trends.
  • Visualizations using Matplotlib to present cumulative engagement by country, category distribution, and individual channel comparisons.
